# ProQuote Cloud

A modern cloud-based quotation and proforma invoice management platform that helps businesses create, manage, and share professional quotations and invoices with ease.

## 🚀 Overview

ProQuote Cloud is designed for freelancers, small businesses, sales teams, and service providers who need a fast and efficient way to generate quotations and proforma invoices. The application provides automated calculations, document management, and a user-friendly dashboard to streamline business operations.

## ✨ Features

### 📄 Document Management
- Create professional quotations
- Generate proforma invoices
- Edit and update existing documents
- Manage document status
- View document history

### 💰 GST & Pricing

### 👥 Customer Management

### 📊 Dashboard

- Easy navigation and document search

### 📱 Responsive Design

### 🔒 Security

## ⚙️ Installation

### Prerequisites

Make sure you have installed:

- Node.js (v18 or later)
- npm or yarn
- Supabase Account

### Clone the Repository

```bash
git clone https://github.com/Shan-Ali4/proquote-cloud.git
```

```bash
cd proquote-cloud
```

### Install Dependencies

```bash
npm install
```

### Configure Environment Variables

Create a `.env` file in the root directory:

```env
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
```

### Start Development Server

```bash
npm run dev
```

Application will be available at:

```bash
http://localhost:5173
```
